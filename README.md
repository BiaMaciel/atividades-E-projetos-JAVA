# ☕ Fundamentos de Java

Este repositório contém atividades e exercícios desenvolvidos durante meus estudos iniciais em **Java**, com foco na prática da lógica de programação e compreensão dos conceitos fundamentais da linguagem.

## 📚 Objetivo

O objetivo deste projeto é reforçar conceitos básicos de programação utilizando Java, desenvolvendo habilidades essenciais para a construção de aplicações.

## 🧠 Conteúdos abordados

- Estrutura básica de um programa em Java  
- Variáveis e tipos de dados  
- Operadores (aritméticos, relacionais e lógicos)  
- Estruturas condicionais (`if`, `else`, `switch`)  
- Estruturas de repetição (`for`, `while`)  
- Entrada de dados com `Scanner`  
- Boas práticas iniciais de código
